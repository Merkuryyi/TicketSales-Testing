## О проекте

В этом репозитории представлен проект системы для тестирования.

## Содержимое

- `test-cases` — файл с тест-кейсам

- `CTT` — файлы проекта

- SFML

## Система, реализующая продажу билетов и проездных в общественном транспорте
Веб-приложение для автоматизации продажи билетов и проездных в общественном транспорте. 
Решение позволяет пассажирам быстро приобретать разовые и месячные проездные билеты, а также получать актуальную информацию о работе транспорта. 

- Использована база данных PostgreSQL.

- C# с фреймворками SFML, Npgsql.

- Интерфейс спроектирован в Figma.

Функции:

1) реализована авторизация и регистрация пользователей с подтверждением по коду;

2) реализована покупка разовых билетов с выбором льготной категории ("Общий", "Пенсионный", "Багажный") и их количества;

3) реализована покупка проездных билетов на месяц с выбором льготной категории ("Общий", "Студенческий", "Пенсионный");

4) реализован просмотр каталога всех доступных билетов и проездных с ценами и описанием;

5) реализован раздел "Транспортные новости" для информирования пользователей;

6) реализован профиль с функцией изменения имени и фамилии, просмотра данных для входа и системой уведомлений;

7) реализована история покупок с отображением последних приобретенных билетов и проездных на главной странице;

8) реализован поиск по ключевым словам для быстрой навигации по разделам приложения;

9) реализован быстрый вход в профиль с использованием JSON-файла для хранения данных сессии.
